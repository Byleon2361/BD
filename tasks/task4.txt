Задание:
1. Настроить репликацию данных для всех используемых таблиц.
2. Настроить автоматическое создание дампа, нужен дамп не только данных, но и настроек БД
3. Сделать копию таблицы с самым большик кличеством записей, для нее настроить партицирование. Провести эксперимент по запросам к этим таблицам (партицированной и нет), с указанием разных условий для выборки (как по условию партицирования, так и нет)
4. Посмотреть разницу между logged и unlogged таблицами. Выполнить замеры на вставку, чтение, удаление (вставлять записи по одной/группой)
5. Реализовать запросы для разработанных кейсов.
P.S. Все результаты замеров сохранить с указанием входных условий.
